# 📊 Sales Performance Dashboard

## 📌 Business Problem
Organizations need clear visibility into sales performance to make informed business decisions.  
This project analyzes sales data across regions, customer segments, and product categories to identify key revenue drivers and growth opportunities.

---

# 📊 Dataset

The dataset includes:

- Order Information  
- Customer Information  
- Geographic Information  
- Product Information  
- Sales Metrics  

---

# 🎯 Objectives

- Analyze sales performance  
- Identify top-performing regions and products  
- Understand customer behavior  
- Detect sales trends over time  

---

# 🧠 Project Workflow

| Step | Description | Tool Used |
|---|---|---|
| 1️⃣ Data Collection | Collected and organized raw sales data. | Excel |
| 2️⃣ Data Cleaning | Removed duplicates, fixed formats, and handled missing values. | Power Query, Excel |
| 3️⃣ Exploratory Analysis | Performed initial analysis and data checks. | Python (Pandas, Matplotlib) |
| 4️⃣ Data Modeling | Created relationships between tables and organized the data model. | Power BI Model View |
| 5️⃣ KPI Development | Built sales metrics and calculations. | DAX |
| 6️⃣ Dashboard Design | Created interactive charts, filters, and visual reports. | Power BI |
| 7️⃣ Insights Analysis | Identified trends, top-performing regions, and business opportunities. | Power BI |

---

# 📈 Key Metrics (KPIs)

- **Total Sales**  
- **Total Orders**  
- **Average Sales per Order**  
- **Sales by Category**  
- **Sales by Region**  

---

# 📊 Dashboard Features

- 📅 Monthly and yearly sales trends  
- 🌍 Regional sales analysis  
- 📦 Product category performance  
- 👥 Customer segmentation  
- 🎛️ Interactive slicers and filters
