# 📊 Sales Performance Dashboard

## 📌 Business Problem
Organizations need clear visibility into sales performance to make informed business decisions.  
This project analyzes sales data across regions, customer segments, and product categories to identify key drivers of revenue and uncover growth opportunities.

---

## 📊 Dataset

This dataset contains transactional sales data, including:

### 🧾 Order Information
- Order ID  
- Order Date  
- Ship Date  
- Ship Mode  

### 👥 Customer Information
- Customer ID  
- Customer Name  
- Segment (Consumer, Corporate, Home Office)

### 🌍 Geographic Information
- Country  
- City  
- State  
- Postal Code  
- Region  

### 📦 Product Information
- Product ID  
- Category  
- Sub-Category  
- Product Name  

### 💰 Metrics
- Sales  

---

## 🎯 Objectives

- Analyze overall sales performance  
- Identify top-performing regions and cities  
- Evaluate product category performance  
- Understand customer segment behavior  
- Detect trends over time  

---

## 🧠 Approach

1. Data cleaning and formatting using Power Query  
2. Creation of date-based columns (Year, Month)  
3. Development of key metrics using DAX  
4. Interactive dashboard design with filters and slicers  
5. Insight generation based on data patterns  

---

## 📈 Key Metrics (KPIs)

- **Total Sales** = SUM(Sales)  
- **Total Orders** = Count of Order ID  
- **Average Sales per Order**  
- **Sales by Category**  
- **Sales by Region**  

---

## 📊 Dashboard Features

- 📅 Time-based analysis (monthly and yearly trends)  
- 🌍 Geographic breakdown (region/state level)  
- 📦 Product performance (category & sub-category)  
- 👥 Customer segmentation analysis  
- 🎛️ Interactive filters for dynamic exploration
